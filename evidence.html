Future Care Network — Childcare Matching Platform

A full-stack childcare matching platform developed as the capstone project for CSC6200 Advanced ICT Professional Project at the University of Southern Queensland. Built for a real client — Angela Cochrane of Future Care Network — the platform addresses a critical gap in regional Australian childcare access.

Families in rural and regional Australia currently rely on unregulated Facebook groups with 50,000+ members to find childcare. Educators face $10,000 startup costs with no structured support. This platform was built to solve both problems with a verified, centralised matching system.

Project Deliverables

✓ Working MVP system for child caer facelity
✓ Family and educator registration with secure login
✓ Admin dashboard with live KPIs and data visualisation
✓ Blue Card compliance tracking
✓ Regional demand heatmap for government funding support
✓ Full documentation and user manual delivered to client
