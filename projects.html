Project 03

Future Care Network

A full-stack childcare matching platform built as part of CSC6200 Advanced ICT Professional Project at the University of Southern Queensland, Trimester 1 2026. The client was Angela Cochrane, representing Future Care Network a regional childcare initiative connecting families with qualified educators across rural and regional Australia.

The core problem: families in regional Australia rely on unregulated Facebook groups with 50,000+ members to find childcare, while educators face $10,000 startup costs with no support and no centralised platform to reach families. Our team built an MVP to solve this directly.

My Role

✓ Contributed to backend development and system architecture using Laravel 11
✓ Contributed to project documentation, user manual, and sprint planning across 8 weeks
✓ Supported deployment to Railway cloud hosting and final client delivery

Key Features

✓ Family and educator registration with secure login system
✓ Admin dashboard with live KPI cards and Chart.js data visualisation
✓ Blue Card compliance tracking for educator verification
✓ Auto-matching algorithm based on postcode and care type
✓ Regional demand heatmap by postcode to support government funding applications

Restaurant Web Application

A full-stack web application built using the Laravel PHP framework for a restaurant business. The system provides a comprehensive admin panel for managing the restaurant's menu, orders, and content.

This project involved designing a normalised MySQL relational database, implementing a secure user authentication system, and building a modern frontend using Tailwind CSS and Vite for asset bundling.

Key Features

✓ Admin panel with full CRUD operations for menu management
✓ Secure authentication and role-based access control
✓ Normalised MySQL database design with Eloquent ORM
✓ Responsive frontend using Tailwind CSS
✓ Asset compilation with Vite for optimised build performance
